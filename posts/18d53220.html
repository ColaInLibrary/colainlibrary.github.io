1 安装

osqp的官方git: https://github.com/osqp/osqp
下载最新的Release版本，发现其中是已经编译好到.so和头文件，但是在包含osqp.h的时候会报错没有osqp_configure.h，而这个文件是从osqp源码构建时编译生成的。。。

osqp-eigen的git:https://github.com/robotology/osqp-eigen
osqp-eigen提供了conda安装，建议优先选择conda安装，因为它会同步处理好相应的osqp及Eigen版本依赖，而又不污染系统环境。

我用conda装好后，仍然会出现一些头文件找不到的问题。环境的问题先暂时告一段落，先用之前已经在系统目录中安装好的来继续开发。

松弛变量的作用

松弛变量在二次规划（QP）中的作用

在二次规划（Quadratic Programming, QP）中引入松弛变量（Slack Variables）主要有以下作用：

1. 将不等式约束转化为等式约束

问题：QP 的标准形式通常要求约束是线性的，但实际问题可能包含不等式约束（如 Ax≤bA x \leq bAx≤b）。
解决：通过引入非负松弛变量 s≥0s \geq 0s≥0，将不等式约束转化为等式约束：

Ax≤b⇒Ax+s=b,s≥0A x \leq b \quad \Rightarrow \quad A x + s = b, \quad s \geq 0 Ax≤b⇒Ax+s=b,s≥0

优点：

  • 统一约束类型，方便使用标准 QP 求解器（如 OSQP）。
  • 避免直接处理不等式约束的复杂性。

示例：

  • 原始约束：x1+x2≤5x_1 + x_2 \leq 5x1​+x2​≤5
  • 引入松弛变量 sss：

    x1+x2+s=5,s≥0x_1 + x_2 + s = 5, \quad s \geq 0 x1​+x2​+s=5,s≥0

2. 处理非线性约束或非凸问题

问题：某些非线性约束（如 ∥x∥≤1\|x\| \leq 1∥x∥≤1）无法直接用 QP 求解。
解决：通过松弛变量将其转化为线性约束：

∥x∥≤1⇒∥x∥2≤1+s,s≥0\|x\| \leq 1 \quad \Rightarrow \quad \|x\|^2 \leq 1 + s, \quad s \geq 0 ∥x∥≤1⇒∥x∥2≤1+s,s≥0

优点：

  • 将非凸问题近似为凸问题，提高求解效率。
  • 适用于 SOCP（Second-Order Cone Programming）等扩展问题。

3. 避免严格不等式约束的数值问题

问题：严格不等式（如 Ax<bA x < bAx<b）在数值计算中可能导致无解或不稳定。
解决：引入松弛变量 s>0s > 0s>0：

Ax≤b−s,s>0A x \leq b - s, \quad s > 0 Ax≤b−s,s>0

优点：

  • 确保约束可行域非空。
  • 提高算法的数值稳定性。

4. 目标函数中的正则化（如软约束）

问题：硬约束（如 Ax=bA x = bAx=b）可能过于严格，导致无解。
解决：引入松弛变量并添加到目标函数中，形成软约束：

min⁡12x⊤Px+q⊤x+λ∥s∥2s.t.Ax+s=b\min \quad \frac{1}{2} x^\top P x + q^\top x + \lambda \|s\|^2 \quad \text{s.t.} \quad A x + s = b min21​x⊤Px+q⊤x+λ∥s∥2s.t.Ax+s=b

优点：

  • 允许轻微违反约束（通过 sss调节），提高问题可行性。
  • 适用于鲁棒优化或机器学习中的正则化。

5. 具体应用场景

(1) 支持向量机（SVM）

  • 松弛变量允许某些样本分类错误，提高模型泛化能力：

    min⁡12∥w∥2+C∑ξis.t.yi(w⊤xi+b)≥1−ξi,ξi≥0\min \quad \frac{1}{2} \|w\|^2 + C \sum \xi_i \quad \text{s.t.} \quad y_i (w^\top x_i + b) \geq 1 - \xi_i, \quad \xi_i \geq 0 min21​∥w∥2+C∑ξi​s.t.yi​(w⊤xi​+b)≥1−ξi​,ξi​≥0

    其中 ξi\xi_iξi​是松弛变量。

(2) 线性规划（LP）的单纯形法

  • 松弛变量将不等式约束转化为等式，便于构造初始可行基。

(3) 模型预测控制（MPC）

  • 松弛变量处理状态或输入的约束违反，保证控制器可行性。

6. 松弛变量的数学形式

标准 QP 问题

原始问题：

min⁡12x⊤Px+q⊤xs.t.Ax≤b\min \quad \frac{1}{2} x^\top P x + q^\top x \quad \text{s.t.} \quad A x \leq b min21​x⊤Px+q⊤xs.t.Ax≤b

引入松弛变量后：

min⁡12x⊤Px+q⊤xs.t.Ax+s=b,s≥0\min \quad \frac{1}{2} x^\top P x + q^\top x \quad \text{s.t.} \quad A x + s = b, \quad s \geq 0 min21​x⊤Px+q⊤xs.t.Ax+s=b,s≥0

变量维度：若原始问题有 nnn 个变量和 mmm 个约束，引入松弛变量后，变量变为 n+mn + mn+m 维。

8. 松弛变量的注意事项

  1. 维度增加：引入松弛变量会增大问题规模，可能影响计算效率。
  2. 非负性：松弛变量通常要求 s≥0s \geq 0s≥0，需在约束中显式指定。
  3. 目标函数惩罚：若松弛变量出现在目标函数中（如正则化），需权衡惩罚系数 λ\lambdaλ。

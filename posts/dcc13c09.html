LeggedGym 安装

发表于2025-04-15|更新于2025-10-29|工科生的基本功软件技术
|浏览量:

1 环境配置

  • 计算机配置：Intel i7-13700, RTX 4060 8GB
  • 软件环境：Ubuntu 20.04 LTS, Python 3.8

2 安装过程

2.1 Pytorch安装

我这里安装的显卡驱动对应的版本是 535.171.04， cuda 版本是12.2。因此需要安装相应版本的pytorch。
最新的 pytorch 要求 python 版本是3.9+。因此需要从早期版本寻找对应的版本：https://pytorch.org/get-started/previous-versions/

2.2 IssacGym安装

下载 IsaacGym Preview 4: https://developer.nvidia.com/isaac-gym/download

2.3 rsl-rl 安装

Clone from https://github.com/leggedrobotics/rsl_rl

2.4 LeggedGym安装

Clone from https://github.com/leggedrobotics/legged_gym?tab=readme-ov-file

sympy 需要升级多半是因为在安装 sympybotics 时指定了 0.7.5 版本。

3 测试LeggedGym

运行后如果能正常出现anamal窗口，则表示安装成功。

有香港的学校对LeggedGym进行了修改，适配IsaacLab。LeggedLab: https://github.com/Hellod035/LeggedLab

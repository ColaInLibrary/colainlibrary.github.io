介绍标准 DH(SDH) 机器人建系方法。

1 前言

在做机器人控制的时候，为了能够方便地对机器人进行力矩控制，自己搭建了一套控制器。在搭建完基本的硬件平台后，需要对机器人进行建模。首先是机器人坐标系的建立。以UR10为例。

有两种可以在 MATLAB 中使用的工具箱：

  • PerterCoke的Robotics ToolBox
  • ARTE: A ROBOTICS TOOLBOX FOR EDUCATION

第一种工具箱对应的有一本书 《Robotics, Vision and Control》[1]，这个用的比较多，不再赘述。

第二种工具箱下载好后将路径添加到 MATLAB 中，然后在它的文件夹下运行一下命令即可。

2 SDH建立坐标系的过程

上图所示的转动关节 DH 参数的定义[1]：

  1. 关节距离 djd_jdj​：从 xj−1x_{j-1}xj−1​ 轴沿 zj−1z_{j-1}zj−1​ 轴到 xjx_jxj​ 轴的距离；
  2. 关节转角 θj\theta _jθj​：从 xj−1x_{j-1}xj−1​ 轴沿 zj−1z_{j-1}zj−1​ 轴转到 xjx_jxj​ 轴的角度，且 θj∈(−π,π]\theta_j \in (-\pi,\pi]θj​∈(−π,π]；
  3. 杆间距离 aja_jaj​：从 zj−1z_{j-1}zj−1​ 轴沿 xjx_jxj​ 轴到 zjz_jzj​ 轴的距离；
  4. 杆件扭角 αj\alpha_jαj​：从 zj−1z_{j-1}zj−1​ 轴沿 xjx_jxj​ 轴转到 zjz_jzj​ 轴的角度，且 αj∈(−π,π]\alpha_j \in (-\pi,\pi]αj​∈(−π,π].

DH 坐标系中轴与原点确定的基本原则是[2]：

  1. zjz_jzj​ 轴沿关节 j+1j+1j+1 的轴向(指向可任选，但通常都将各平行的 zzz 轴均取为相同的指向)；
  2. 原点选取的原则：原点 OjO_jOj​ 为 zj−1z_{j-1}zj−1​ 轴和 zjz_jzj​ 轴的公法线与 zj−1z_{j-1}zj−1​ 轴的交点.
    • 当 zj−1z_{j-1}zj−1​ 轴和 zjz_jzj​ 轴平行时，两轴之间有无数条公垂线，若两轴重合，取 Oj=Oj−1O_j = O_{j-1}Oj​=Oj−1​；若两轴平行但不重合，则过 Oj−1O_{j-1}Oj−1​ 点作 zj−1z_{j-1}zj−1​ 轴和 zjz_jzj​ 轴的公垂线，此公垂线与 zjz_jzj​ 轴的交点即 OjO_jOj​.
    • 由于没有 z−1z_{-1}z−1​，所以上述方法不适用于 O0O_0O0​，这时确定 O0O_0O0​ 的方法是：若 z0z_0z0​ 与 z1z_1z1​ 相交，取 O0=O1O_0 = O_1O0​=O1​；若z0z_0z0​ 与 z1z_1z1​ 不相交，取 O0O_0O0​ 在 z0z_0z0​ 与 z1z_1z1​ 的公垂线上。
  3. 轴 xjx_jxj​ 与轴 zj−1z_{j-1}zj−1​ 相交且垂直，即 xjx_jxj​ 为沿过 zj−1z_{j-1}zj−1​ 轴和 zjz_jzj​ 轴的公垂线，方向从 zj−1z_{j-1}zj−1​ 指向 zjz_jzj​.
    • 当 zj−1z_{j-1}zj−1​ 轴和 zjz_jzj​ 轴重合时，这时 Oj=Oj−1O_j = O_{j-1}Oj​=Oj−1​，那么在零位时要求 xj=xj−1x_j = x_{j-1}xj​=xj−1​.
    • 当 zj−1z_{j-1}zj−1​ 轴和 zjz_jzj​ 轴相交且不重合时，选择 xj=±(zj−1×zj)x_j = \pm(z_{j-1}\times z_j)xj​=±(zj−1​×zj​)，通常使所有平行的 xxx 轴均具有相同的指向.
    • 当 j=0j = 0j=0时，O0=O1O_0 = O_1O0​=O1​, 或 O0O_0O0​ 在 z0z_0z0​ 轴和 z1z_1z1​ 轴的公垂线上，选取 x0x_0x0​ 轴与 x1x_1x1​ 轴重合.

Tip: 上述方法没有考虑平动关节的建系，可参见[2]。

    • 从 z1z_1z1​ 轴沿 x2x_2x2​ 轴到 z2z_2z2​ 轴的距离为 a2a_2a2​，即 连杆 222 的长度；
    • 因 z1//z2z_1 // z_2z1​//z2​，从 z1z_1z1​ 轴沿 x2x_2x2​ 轴转到 z2z_2z2​ 轴的角度为 α2=0\alpha_2 = 0α2​=0.
  3. j=3j = 3j=3 时，注意连杆 333 实际上由电机所在水平段与包含 444 关节在内的竖直段组成：
    • 从 x2x_2x2​ 轴沿 z2z_2z2​ 轴到 x3x_3x3​ 轴的距离为 −d3-d_3−d3​，即连杆 333 水平段长度的相反数；
    • 因 x2//x3x_2 // x_3x2​//x3​，从 x2x_2x2​ 轴沿 z2z_2z2​ 轴转到 x3x_3x3​ 轴的角度为 θ3=0\theta_3 = 0θ3​=0；
    • 从 z2z_2z2​ 轴沿 x3x_3x3​ 轴到 z3z_3z3​ 轴的距离为 a3a_3a3​，即 连杆 333 竖直段的长度；
    • 因 z2//z3z_2 // z_3z2​//z3​，从 z2z_2z2​ 轴沿 x3x_3x3​ 轴转到 z3z_3z3​ 轴的角度为 α3=0\alpha_3 = 0α3​=0.
  4. j=4j = 4j=4 时，
    • 从 x3x_3x3​ 轴沿 z3z_3z3​ 轴到 x4x_4x4​ 轴的距离为 d4d_4d4​，即连杆 444 长度；
    • 因 x3⊥x4x_3 \perp x_4x3​⊥x4​ 且不共面，从 x3x_3x3​ 轴沿 z3z_3z3​ 轴转到 x4x_4x4​ 轴的角度为 θ4=−π/2\theta_4 = -\pi/2θ4​=−π/2；
    • 因 z3⊥z4z_3 \perp z_4z3​⊥z4​ 且共面，从 z3z_3z3​ 轴沿 x4x_4x4​ 轴到 z4z_4z4​ 轴的距离为 a4=0a_4 = 0a4​=0；
    • 从 z3z_3z3​ 轴沿 x4x_4x4​ 轴转到 z4z_4z4​ 轴的角度为 α4=−π/2\alpha_4 = -\pi/2α4​=−π/2.
  5. j=5j = 5j=5 时，
    • 从 x4x_4x4​ 轴沿 z4z_4z4​ 轴到 x5x_5x5​ 轴的距离为 d5d_5d5​，即连杆 555 长度；
    • 因 x4//x5x_4 // x_5x4​//x5​，从 x4x_4x4​ 轴沿 z4z_4z4​ 轴转到 x5x_5x5​ 轴的角度为 θ5=0\theta_5 = 0θ5​=0；
    • 因 z4⊥z5z_4 \perp z_5z4​⊥z5​ 且共面，从 z4z_4z4​ 轴沿 x5x_5x5​ 轴到 z5z_5z5​ 轴的距离为 a5=0a_5 = 0a5​=0；
    • 从 z4z_4z4​ 轴沿 x5x_5x5​ 轴转到 z5z_5z5​ 轴的角度为 α5=π/2\alpha_5 = \pi/2α5​=π/2.
  6. j=6j = 6j=6 时，
    • 从 x5x_5x5​ 轴沿 z5z_5z5​ 轴到 x6x_6x6​ 轴的距离为 d6d_6d6​，即连杆 666 长度；
    • 因 x5//x6x_5 // x_6x5​//x6​，从 x5x_5x5​ 轴沿 z5z_5z5​ 轴转到 x6x_6x6​ 轴的角度为 θ6=0\theta_6 = 0θ6​=0；
    • 因 z5z_5z5​ 与 z6z_6z6​ 重合，从 z5z_5z5​ 轴沿 x6x_6x6​ 轴到 z6z_6z6​ 轴的距离为 a6=0a_6 = 0a6​=0；
    • 从 z5z_5z5​ 轴沿 x6x_6x6​ 轴转到 z6z_6z6​ 轴的角度为 α6=0\alpha_6 = 0α6​=0.

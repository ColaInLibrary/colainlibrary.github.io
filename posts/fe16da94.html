记录rsl-rl库的学习笔记。

1 rsl-rl库结构框架

  • rsl-rl
    • init.py
    • algorithms
      • init.py
      • PPO.py # PPO算法的实现
    • env
      • init.py
      • vec_envv.py # 实现并行处理多个环境的向量化环境
    • modules
      • init.py
      • actor_critic.py # 定义 Actor-Critic 网络结构
      • actor_critic_recurrent.py # 定义含循环层的 Actor-Critic 网络结构
      • normalizer.py # 数据正规化工具，有助于训练过程的稳定性
    • runners
      • init.py
      • on_policy_runner.py # 实现用于执行 on-policy 算法训练循环的运行器
    • storage
      • init.py
      • rollout_storage.py # 存储和管理策略 rollout 数据的工具
    • utils
      • init.py
      • utils.py # 通用工具类实现函数

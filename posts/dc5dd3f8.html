Motive 软件使用方法

发表于2021-12-19|更新于2022-08-18|工科生的基本功
|浏览量:

简要记录Motive的使用方法。

1 前言

最近在用动捕系统做实验，简单记录下motive的相关使用方法，防止记忆丢失。

2 所用相机

所用相机：OptiTrack V120:Trio, 官网链接
相机的硬件连接方式：链接

Tip1: 都是标准化的接口，直接照图插上就能用。不要使用250V的圆孔电源插头。
Tip2: 该款三目相机无需进行标定，但是坐标原点的位置在第一个镜头和第二个镜头之间偏后约30cm的位置。
Tip3: 相关的参考文档都能从官网中找到。

3 软件

软件为 motive2.3.0. 下载链接
配套的SDK，NaNet SDK4.0.0. 下载链接

我的目的是测试通软硬件，最后在Linux上读数据；官网上说 Camera SDK 2.3.0 与 Motive2.3.0 相匹配，尚未测试。

Tip1: 软件中地面的一小格为20cm，一大格为1m。
Tip2: 将 View -> Data Streaming Pane 中的所有开关都打开。
Tip3: 建造刚体： View -> Builder Pane, 选中marker点，输入名字，点击 Create。
读取数据的方式：
运行 NatNet_SDK_4.0\NatNetSDK\Samples\bin\x64\SampleClient.exe

4 Ubuntu读取

  1. 从 下载链接 获取配套的NaNet SDK4.0.0. Linux 示例程序。
  2. 用一根网线将Ubuntu和Windows连接。
  3. 关闭Windows的防火墙。
  4. 设置IP地址，并测试双向是否ping通。
  5. Motive 中打开 View -> Data Streaming Pane 面板，将 Local Interface： 改为第4步设置的Windows IP。
  6. Ubuntu 按照SDK中的说明运行程序。 ./SampleClient WindowsIP

结果：
读取到与第3节中一致的结果。

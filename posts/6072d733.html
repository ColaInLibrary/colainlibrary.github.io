Mathematics for Machine Learning：【06】概率论基础知识

1 期望

1.1 离散随机变量期望

E[X]=∑i=1nxip(xi)\mathbb{E}[X] = \sum_{i=1}^{n} x_i p(x_i) E[X]=i=1∑n​xi​p(xi​)

如果 XXX 的所有取值都是等可能的（均匀分布），即 p(xi)=1np(x_i) = \frac{1}{n}p(xi​)=n1​，则期望退化为算术平均值：

E[X]=1n∑i=1nxi\mathbb{E}[X] = \frac{1}{n} \sum_{i=1}^{n} x_i E[X]=n1​i=1∑n​xi​

期望描述的是概率加权和，是在描述理论上的长期（无限数据）平均值，它依赖概率分布 p(xi)p(x_i)p(xi​)。而算术平均假设有限数据的所有数据点权重相同，描述有限样本的平均值。

期望的含义是用确定性来描述随机变量不确定性的。
示例：有10个数，其中8个是1，2个是2.显然，随机抽一个数，抽到1的概率为0.8，2的概率为0.2，那么期望就是10.8+20.2=1.6。

1.2 连续随机变量期望

在强化学习中，**动作价值函数（Action-Value Function, Q-function）和状态价值函数（State-Value Function, V-function）**是两类核心的价值函数，它们共同用于评估策略的优劣，但在定义、用途和计算方式上存在显著差异和联系。

2. 状态价值函数和动作价值函数的区别与联系

2.1 定义与核心区别

状态价值函数 ( V^\pi(s) )

• 定义：在状态 ( s ) 下，遵循策略 ( \pi ) 的长期期望回报（即累积奖励的期望值）。
[
V^\pi(s) = \mathbb{E}\pi \left[ \sum{k=0}^\infty \gamma^k r_{t+k} \mid s_t = s \right]
]
• 特点：
• 仅依赖状态 ( s )，不涉及具体动作。
• 反映策略 ( \pi ) 下状态的“好坏”。

动作价值函数 ( Q^\pi(s, a) )

• 定义：在状态 ( s ) 下执行动作 ( a ) 后，继续遵循策略 ( \pi ) 的长期期望回报。
[
Q^\pi(s, a) = \mathbb{E}\pi \left[ \sum{k=0}^\infty \gamma^k r_{t+k} \mid s_t = s, a_t = a \right]
]
• 特点：
• 同时依赖状态 ( s ) 和动作 ( a )。
• 用于评估在特定状态下采取特定动作的价值。

2.2 联系

(1) 通过策略 ( \pi ) 关联

• ( V^\pi(s) ) 是 ( Q^\pi(s, a) ) 关于动作 ( a ) 的期望：
[
V^\pi(s) = \sum_{a} \pi(a \mid s) Q^\pi(s, a)
]
即，状态价值是策略下所有可能动作价值的加权平均。

(2) 贝尔曼方程

• ( V^\pi(s) ) 的贝尔曼方程：
[
V^\pi(s) = \sum_{a} \pi(a \mid s) \sum_{s’} P(s’ \mid s, a) \left[ r(s, a, s’) + \gamma V^\pi(s’) \right]
]
• ( Q^\pi(s, a) ) 的贝尔曼方程：
[
Q^\pi(s, a) = \sum_{s’} P(s’ \mid s, a) \left[ r(s, a, s’) + \gamma \sum_{a’} \pi(a’ \mid s’) Q^\pi(s’, a’) \right]
]
两者均通过动态规划递归关联。

(3) 最优价值函数的关系

• 最优策略 ( \pi^* ) 下，两者满足：
[
V^(s) = \max_a Q^(s, a), \quad Q^(s, a) = r(s, a) + \gamma \sum_{s’} P(s’ \mid s, a) V^(s’)
]

2.3 关键区别

特征 状态价值函数 ( V^\pi(s) ) 动作价值函数 ( Q^\pi(s, a) )
输入变量 仅状态 ( s ) 状态 ( s ) 和动作 ( a )
用途 评估策略 ( \pi ) 下状态的整体价值 评估策略 ( \pi ) 下特定动作的价值
策略改进 无法直接用于选择动作（需结合模型） 可直接选择最优动作（如 ( \arg\max_a Q )）
模型依赖性 依赖环境模型 ( P(s’ \mid s, a) ) 可模型无关（如Q-learning）

2.4 直观理解

• ( V^\pi(s) )：类似于“当前位置的得分”，不关心具体动作。
• ( Q^\pi(s, a) )：类似于“在当前位置选择某条路后的得分”，直接指导决策。

2.5 应用场景

• ( V^\pi(s) ) 的用途：
• 策略评估（如Policy Iteration）。
• 模型已知时用于规划（如动态规划）。
• ( Q^\pi(s, a) ) 的用途：
• 无模型学习（如Q-learning、DQN）。
• 直接生成贪婪策略（( \pi(s) = \arg\max_a Q(s, a) )）。

2.6 总结

• 联系：两者通过策略 ( \pi ) 和贝尔曼方程相互关联，且最优策略下 ( V^* ) 和 ( Q^* ) 可互相推导。
• 区别：( V ) 函数评估状态，( Q ) 函数评估动作，后者更直接适用于控制问题。在无模型强化学习中，( Q ) 函数更为常用。
